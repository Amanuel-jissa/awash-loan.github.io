# Awash Loan Calculator Site Bundle

Files:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

This is a static website. You can deploy it to any static host.

## Quick deploy options
1. Netlify: drag this folder or zip onto Netlify's deploy page.
2. GitHub Pages: upload these files to a GitHub repository and enable Pages.
3. Cloudflare Pages: create a Pages project and upload these static files.

## Notes
- Works on Android phones in the browser.
- Can be added to the home screen.
- After the first load, the service worker caches the app for offline use.
